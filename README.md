# 💰 MoneyFlow AI
Aplicación de gestión de gastos personales con asistencia de Inteligencia Artificial (Gemini).

## 🚀 Características
- **Registro de Gastos:** Guarda tus gastos localmente en un archivo JSON.
- **IA Guru:** Un coach financiero con humor que analiza tus finanzas.
- **Persistencia:** Tus datos no se borran al cerrar la app.
- **Navegación Moderna:** Interfaz fluida con NavigationBar.

## 🛠️ Instalación
1. Clona el repositorio.
2. Instala las librerías: `pip install flet google-genai python-dotenv`.
3. Crea un archivo `.env` con tu `GEMINI_API_KEY`.
4. Ejecuta con: `python main.py`.
